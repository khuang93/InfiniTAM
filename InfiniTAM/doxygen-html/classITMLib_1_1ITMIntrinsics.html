bool ITMLib::ITMIntrinsics::FocalLengthSignsDiffer ( ) const
inline

Returns true if the two focal lengths have a different sign.

Note
This is used to handle datasets such as ICL_NUIM and other non standard inputs where one of the two focal lengths is negative: that causes the normals to point away from the camera. This causes valid points to be ignored during visualisation and tracking.

The problem presents itself only when computing normals as cross product of the difference vectors between raycasted points and is thus solved by flipping the normal direction.
