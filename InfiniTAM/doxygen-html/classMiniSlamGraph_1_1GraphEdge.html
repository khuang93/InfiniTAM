virtual bool MiniSlamGraph::GraphEdge::computeJacobian ( const NodeIndex &  nodes,
int  id,
double *  j 
) const
pure virtual

This method is supposed to compute the Jacobian of the Edge/Contraint. Here the "Jacobian" means the derivative of the residual vector from computeResidualVector() w.r.t. the parameters of the Node with id id. The result is stored in a row-major "matrix" j, where each row contains the derivatives of a single entry of the residual vector w.r.t. all the parameters. This means the derivative of residual i_r w.r.t. parameter i_p is addressed as j[i_r * dimMeasure + i_p] . If the node identified py id does not affect the residual, j is untouched and the return value is set to false. In all other cases, the return value is set to true.
